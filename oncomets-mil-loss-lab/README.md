# oncomets-mil-loss-lab

Pilot skill for OncoMETS to iterate on CLAM/MIL losses under class imbalance and weak-to-semi-supervised transitions.

## What this gives you
- A pragmatic, low-friction workflow for loss selection and evaluation.
- A minimal experiment matrix suitable for part-time research cadence.
- Explicit go/no-go and rollback criteria to avoid drift.

## Structure
- SKILL.md: main workflow and activation guidance.
- references/loss-playbook.md: formulas and practical PyTorch notes.
- scripts/: reserved for future automation.
- assets/: reserved for future diagrams and templates.
